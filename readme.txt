This is simple WP Page Builder addon for WooComerce.

== Description ==

WP Page Builder has no built in WooComerce addon. This is simple WP Page Builder addon for WooComerce. You can make a beafutiful shop page with this plugin.

== Installation ==

Search on the plugins "WP Page Builder WooCommerce Addon" and install/activate.

or

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
